Einführung

Beginnen Sie mit einer erstklassigen CSS-Bibliothek, die verwendet wurde, um coolunicorn und alle möglichen anderen Projekte zu erstellen. Genau hier. Auf dieser Dokumentenseite.



Schnellstart

Kopiere diesen 
<link>
und füge ihn in deinem 
<head>
vor allen anderen Stylesheets ein um das CSS zu laden. 
<link href="https://coolunicorn.ml/style.css" rel="stylesheet" type="text/css" />

Oder lade dir diese Datei herunter!


Starter Template

Stellen Sie sicher, dass Ihre Seiten mit den neuesten Design- und Entwicklungsstandards eingerichtet sind. Das bedeutet, dass Sie einen HTML5-Dokumenttyp verwenden und ein Viewport-Meta-Tag für ein angemessenes Schnelles-Verhalten einfügen. Füge alles zusammen ein und deine Seiten sollten so aussehen: 
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width">
    <title>CoolUnicorn.css</title>
    <link href="https://coolunicorn.ml/style.css" rel="stylesheet" type="text/css" />
  </head>
  <body>
    <nav class="nav">
      <div class="nav-item">
        <img src="https://coolunicorn.ml/icon.png" class="nav-img">
      </div>
      <div class="nav-item">
        <a class="nav-link" href>CoolUnicorn.css</a>
      </div>
      <div class="nav-item">
        <button class="nav-button"  onClick="location.replace('https://de.coolunicorn.ml/docs/')">Los gehts</button>
      </div>
    </nav>
    <div class="container floating">
      <h1 class="title">CoolUnicorn.css</h1>
      <p class="subtitle">Schnell, Reagiert und<br> vor allem sieht es gut aus.</p>
      <button class="big-button" onClick="location.replace('https://de.coolunicorn.ml/docs/')">Los gehts</button>
    </div>
    <script src="script.js"></script>
  </body>
</html>

Navigationsleiste

Navigationsleisten sind eines der wichtigsten Elemente einer Website. CoolUnicorn.css macht es einfach, eine Navigationsleiste zu erstellen. 
<nav class="nav">
  <div class="nav-item">
    <a href="../">
    <img src="https://coolunicorn.ml/icon.png" class="nav-img">
    </a>
  </div>
  <div class="nav-item">
    <a class="nav-link" href>CoolUnicorn.css</a>
  </div>
  <div class="nav-item">
    <button class="nav-button"  onClick="location.replace('https://de.coolunicorn.ml/docs/')">Los gehts</button>
  </div>
</nav>
